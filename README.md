# medora-showcase
AI Powered Healthcare Platform built using React Native, FastAPI and Cloud AI services.


Built and maintained by Deepak Jangir.

---

## Overview

MEDORA is a full-stack AI-powered healthcare platform designed to modernize doctor-patient interaction through smart appointment booking, medical report management, and AI-based prescription OCR processing.

The platform enables users to securely book appointments, upload reports, receive notifications, and process prescriptions using cloud-based AI services.
